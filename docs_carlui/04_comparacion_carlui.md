# Comparación de marcos regulatorios

| Criterio                  | Ley 21.459 (Chile)             | Convenio de Budapest                            | GDPR                           |
| ------------------------- | ------------------------------ | ----------------------------------------------- | ------------------------------ |
| Objetivo principal        | Sancionar delitos informáticos | Cooperación internacional contra el cibercrimen | Protección de datos personales |
| Alcance                   | Nacional                       | Internacional                                   | Unión Europea                  |
| Protección de datos       | Indirecta                      | Parcial                                         | Muy alta                       |
| Investigación de delitos  | Sí                             | Sí                                              | No es su enfoque principal     |
| Cooperación internacional | Limitada                       | Alta                                            | Moderada                       |
| Sanciones                 | Penales                        | Coordinación legal                              | Administrativas y económicas   |

## Análisis

La Ley 21.459 permite perseguir penalmente a quienes realizan ataques informáticos. El Convenio de Budapest facilita la cooperación entre países para investigar incidentes transnacionales. Por su parte, el GDPR se enfoca principalmente en la protección de datos personales y la responsabilidad de las organizaciones que los administran.

En conjunto, estos marcos normativos permiten abordar el caso Sony Pictures desde diferentes perspectivas jurídicas y técnicas.
