# Protección de datos personales

## Datos personales comprometidos

Durante el incidente se filtraron nombres, correos electrónicos, direcciones, salarios, contratos y antecedentes laborales de empleados.

## Datos sensibles

Parte de la información expuesta incluía antecedentes privados cuyo uso indebido podía afectar la privacidad de las personas involucradas.

## Derechos ARCO

Las personas afectadas deben poder ejercer sus derechos de:

* Acceso
* Rectificación
* Cancelación
* Oposición

## Relación con la Ley 19.628

La Ley 19.628 busca proteger la información personal y establecer obligaciones para quienes administran bases de datos, principios que resultan relevantes para analizar este incidente.
